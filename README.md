# Калькулятор геометрических фигур

Программа позволяет вычислять площадь и периметр и т.п. для следующих фигур:
- Круг
- Треугольник
- Квадрат

## Технологии
- Язык: Python 3.11
- Парадигма: Объектно-ориентированное программирование (ООП)

## Структура кода
- `shapes.py` — содержит абстрактный класс `Shape` и реализации: `Circle`, `Triangle`, `Square`
- Каждая фигура:
  - Проверяет корректность входных данных
  - Реализует методы `area()` и `perimeter()`

## Пример использования
```python
from shapes import Circle, Triangle, Square

c = Circle(5)
print(f"Круг: площадь = {c.area():.2f}, периметр = {c.perimeter():.2f}")

t = Triangle(3, 4, 5)
print(f"Треугольник: площадь = {t.area():.2f}, периметр = {t.perimeter():.2f}")

s = Square(6)
print(f"Квадрат: площадь = {s.area():.2f}, периметр = {s.perimeter():.2f}")
